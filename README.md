# Ascensao-do-Necromante--Site
# Projeto: Monitor de Conteúdo Web

Atividade avaliativa da faculdade FIAP, focada em desenvolvimento de **HTML, CSS e JavaScript puro**.

## Descrição
O projeto consiste em um site estruturado com **componentização**, aproveitando conhecimentos adquiridos em outra faculdade (ADS) e em projetos realizados com React.  

O site foi desenvolvido inicialmente para monitores de tamanho padrão **1920x1080**, e em próximas versões será implementada maior responsividade para dispositivos móveis.

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript (ES6+)
- Web Components (para componentização)

## Limitações
- Não há backend implementado, pois não é foco desta fase do curso.
- Responsividade ainda não totalmente implementada.

## Próximos passos
- Implementar responsividade para diferentes tamanhos de tela.
- Adicionar backend para armazenar dados do usuário e formulários.
- Melhorar acessibilidade e performance do site.
